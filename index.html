O Processing Community Day é um realizado no mundo todo com o objetivo de promover a interação e celebrar a diversidade de pessoas e tecnologias no universo das artes computacionais.

Esse ano a comunidade brasileira organizou o primeiro PCD nacional, com programação de uma semana inteira.

Programando Coletivamente o Encontros Digitais

Carlos de Oliveira Junior (Vamoss)

Esse é um esforço coletivo para adicionar/alterar/corrigir funcionalidades no Encontros Digitais. Nesta atividade vamos aprender como instalar o Encontros Digitais para rodar no seu computador. A partir disso, vamos aprender como alterar o código e enviar contribuições para melhorar a plataforma.

Público: Todos os públicos
